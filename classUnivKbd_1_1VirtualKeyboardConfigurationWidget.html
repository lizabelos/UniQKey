Detailed Description

Represents a virtual keyboard configuration widget.

The VirtualKeyboardConfigurationWidget class provides a customizable widget that can be used to configure the virtual keyboard.

You don't have to instantiate this class yourself, as it is automatically created by the VirtualKeyboard class.
